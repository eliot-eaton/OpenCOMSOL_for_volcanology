# Published Models

This directory contains peer-reviewed and published COMSOL models related to volcanology.

## Submission Criteria

Models in this directory should:
- Be associated with a published scientific paper
- Include complete documentation
- Be fully validated and tested
- Include all necessary input files and parameters

## Directory Structure

Each model should be placed in its own subdirectory with the following structure:

```
model_name/
├── README.md              # Model description and citation
├── model_file.mph         # COMSOL model file
├── parameters/            # Parameter files
├── results/               # Example results
└── documentation/         # Additional documentation
```

## Naming Convention

Please use descriptive names for your model directories, such as:
- `magma_chamber_deformation_2024`
- `volcanic_degassing_conduit_2023`
- `caldera_subsidence_mechanics_2025`

Include the publication year in the directory name when possible.
